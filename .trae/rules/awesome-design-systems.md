## Usage

When designing UI components, layouts, or visual elements, reference the appropriate DESIGN.md files in the `awesome-design-md/design-md/` directory for inspiration and design patterns.

For example:
- For landing page design patterns → check `vercel/DESIGN.md` or `linear.app/DESIGN.md`
- For dashboard/enterprise UI → check `stripe/DESIGN.md` or `sentry/DESIGN.md`
- For consumer app design → check `spotify/DESIGN.md` or `notion/DESIGN.md`
- For AI product design → check `claude/DESIGN.md` or `cursor/DESIGN.md`
